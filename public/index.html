Change this approval

PATCH /approval/{id}

A JsonPatch array with the fields to be changed. Current supported fields are: status, child, features

JsonPatch
Request Example
[
  {
    "op": "replace",
    "path": "/status",
    "value": "APPROVED"
  }
]
200 OK

Return the changed approval

Schema Definitions

App:

id: integer (int64)

The App Id returned by the system

name: string

The Name of this app

platforms: object[]

An array of enums that specify what platforms are supported by this app.

description: string

A description of the App

logo: string (url)

The url for the logo of this app

urlName: string

The partial url name of this app on Saferize

details: string[]

Features that this app have.

email: string

The app developer's email.

Example
{
  "id": 1234,
  "name": "My App",
  "platforms": [
    "ANDROID",
    "IOS"
  ],
  "description": "My first App",
  "logo": "https://myfirstapp.com/logo.png",
  "urlName": "my-first-app",
  "details": [
    "PAID_APP",
    "SUBSCRIPTION"
  ],
  "email": "dev@myapp.com"
}

Family:

id: integer (int64)

The family Id returned by the system

name: string

The family name

Example
{
  "id": 12345,
  "name": "The Smiths"
}

Parent:

id: integer (int64)

Parent's Id returned by the system

firstName: string

First Name

lastName: string

Last Name

mobilePhone: string (phone)

Mobile Phone. The format should be XXXYYYZZZZ (no dashes or parenthesis)

family: object
email: string (email)
Example
{
  "id": 12345,
  "firstName": "John",
  "lastName": "Smith",
  "mobilePhone": 5551231234,
  "family": {
    "id": 12345,
    "name": "The Smiths"
  },
  "email": "dad@myfamily.com"
}

Child:

id: integer (int64)

The Child Id returned by the system

firstName: string

Child's first name

lastName: string

Child's last Name

birthDate: string (date)

The childs date of birth

family: object
gender: string MALE, FEMALE, UNKNOWN

The child's gender

Example
{
  "id": 12345,
  "firstName": "Sofia",
  "lastName": "Smith",
  "birthDate": "2009-01-23",
  "family": {
    "id": 12345,
    "name": "The Smiths"
  },
  "gender": "FEMALE"
}

AppUser:

id: integer (int64)

The AppUser Id returned by the system

appFeatures: string[]

An array of enums that represent what features the parent allowed this user on this app.

token: string

A client token for the app. This is defined by the app and should be unique per user of the app.

app: object

family: object

The time and date when this approval was created
